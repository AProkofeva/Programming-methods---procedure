Лабораторные по методам программирования - процедурный стиль:
1. Обобщенный артефакт, используемый в задании - квадратные матрицы с целочисленными элементами. 
2. Контейнер организован на основе двунаправленного линейного списка. 
3. Клиентская часть программы размещена в отдельном модуле. Основная часть кода, определяющего реализацию функций или методов, содержится в другом модуле. Программные объекты, обеспечивающие связь между основной и клиентской частями (абстрактные типы данных, сигнатуры функций, описания классов), размещается в отдельном интерфейсном модуле (заголовочном файле). 
4. Обобщение оганизовано на основе использования общей основе (базовой структуры), которая повторяется в каждой из отдельно прописанных специализаций.
